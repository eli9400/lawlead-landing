# LawLead Landing Page

דף נחיתה RTL למכירת מערכת לידים חכמה למשרדי דיני משפחה, עם המרות ישירות לוואטסאפ.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4

## Environment Variables

יש ליצור קובץ `.env.local` לפי `.env.example`:

```bash
NEXT_PUBLIC_WHATSAPP_NUMBER=972501234567
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Deploy

הפרויקט מוכן לפריסה ב־Vercel ללא Backend נוסף.
